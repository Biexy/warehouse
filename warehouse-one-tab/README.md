# Warehouse One Tab

النسخة العربية ذات مساحة العمل الواحدة لنظام إدارة المخزون.

## المزايا

- تسجيل دخول وجلسات منفصلة مع أدوار `ADMIN` و`STOREKEEPER` و`AUDITOR`.
- إدخال وصرف المخزون مع عرض الرصيد الحالي والمتوقع.
- جدول أصناف مرقّم: 10 صفوف افتراضيًا مع خيارات 10 و25 و50.
- تعديل الصنف وتصحيح الحركة داخل نوافذ دون نقل موضع الصفحة.
- سجل تدقيق، CSV، PDF عربي، ونسخ احتياطية حقيقية بصيغة `.xlsx`.
- استيراد أصناف من ملفات `.xlsx` و`.xls` و`.csv` بعد معاينة وفحص كاملين.

## الرفع إلى Google Apps Script

1. انسخ `.clasp.json.example` إلى `.clasp.json`.
2. ضع معرّف مشروع Apps Script الحقيقي بدل القيمة التجريبية.
3. نفّذ `clasp status` وتأكد أن الملفات الظاهرة هي ملفات التشغيل فقط.
4. نفّذ `clasp push`، ثم أنشئ إصدارًا جديدًا من نشر Web App.

يمكن كذلك نسخ ملفات `.gs` و`Index.html` و`App.html` يدويًا إلى مشروع Apps Script. اختبر الاستيراد باستخدام `../sample-data/catalog-76.csv`؛ تُحفظ الأصناف في ورقة `Items`.

لا ترفع ملف `.clasp.json` الحقيقي إلى GitHub لأنه يحتوي معرّف مشروعك.
