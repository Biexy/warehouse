# نظام إدارة المخزون

يحتوي المستودع على نسختين من واجهة Google Apps Script تشتركان في منطق الخادم وقواعد المخزون:

- `warehouse-one-tab`: مساحة عمل عربية واحدة.
- `warehouse-multitab`: واجهة منظمة في خمسة تبويبات.

## استيراد الأصناف

يستطيع المدير رفع ملف `.xlsx` أو `.xls` أو `.csv` من داخل النظام. يعرض النظام معاينة تتضمن الصفوف الجديدة والموجودة والمتعارضة والخاطئة، ولا يكتب إلى Google Sheets قبل الاعتماد النهائي. تُحفظ الأصناف الجديدة في ورقة `Items`، بينما تبقى ورقة الحركات مخصصة للإدخال والصرف والعكس والتصحيح.

الأعمدة المطلوبة:

`الكود، الاسم، المالك، وحدة القياس، الرصيد الافتتاحي، حد إعادة الطلب`

يوجد ملف اختبار كامل في `sample-data/catalog-76.csv`.

## التشغيل

اختر نسخة واحدة فقط، ثم اتبع ملف `README.md` الموجود داخل مجلدها. لكل نسخة إعداد `clasp` مستقل حتى لا تختلط ملفات المشروعين.

## التحقق

شغّل اختبارات Node الموجودة في `tests/*-verify.mjs`. تتحقق الاختبارات من الدخول، الصلاحيات، المخزون، الحركات، الاستيراد، التقارير، وتطابق باكند النسختين.
